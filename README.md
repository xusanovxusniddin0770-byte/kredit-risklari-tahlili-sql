#  Kredit Portfeli va Risk Tahlili (SQL)

##  Loyiha haqida
Ushbu loyiha bank va mikromoliya tashkilotlari uchun **Kredit Portfeli Analitikasi** modelini taqdim etadi. Loyihaning asosiy maqsadi — SQL so'rovlari yordamida muammoli kreditlarni aniqlash va portfel sifatini tahlil qilishdir.

##  Asosiy funksiyalar
- **PAR 30 Tahlili:** 30 kundan ortiq kechikkan kreditlar ulushini hisoblash.
- **Mijozlar Segmentatsiyasi:** Mijozlarni to'lov intizomiga qarab guruhlash.
- **Avtomatlashtirilgan Hisobot:** Kredit qaytarilishi bo'yicha kundalik tahliliy xulosalar.

##  Loyiha tuzilishi
- `1_bazani_yaratish.sql`: Ma'lumotlar bazasi va jadvallar arxitekturasi.
- `2_test_malumotlari.sql`: Loyihani tekshirish uchun realistik mijozlar bazasi.
- `3_risk_tahlili_kodlari.sql`: Riskni hisoblovchi asosiy SQL so'rovlari.

##  Qanday ishlatiladi?
Ushbu SQL skriptlarni istalgan ma'lumotlar bazasi muhitida (PostgreSQL, MySQL, SQLite) ishga tushirish va natijalarni tahlil qilish mumkin.
